About the Project 
Customer churn prediction is the process of identifying which customers are likely to stop 
using a product or service. Accurate churn prediction helps businesses reduce churn costs by 
identifying at-risk customers early and taking proactive retention actions.  
This application takes user inputs related to customer attributes and feeds them into a pre
trained machine learning model to determine the likelihood of churn. 
пїЅ
пїЅ How It Works 
1. Input Form: Users fill in customer details such as demographic and service 
information. 
2. Machine Learning Model: The application uses a classification model trained on 
historical churn data. It may use techniques such as one-hot encoding for categorical 
features and appropriate scaling for numeric values.  
3. Prediction: The model outputs whether the customer is likely to churn and displays 
the probability.
